## 如果文档被打乱应如何修复

如果遇到表格被移到标题前面，或说明文本顺序错乱的情况，修复步骤：

1. **识别各部分内容**
   - 标题：`### 3. 第三阶段：中期检查`
   - 说明段落：以"中期检查主要..."开头
   - 前导句："建议中期检查前准备如下材料："
   - 表格：从 `| 材料名称 |` 开始的 markdown 表格

2. **重新排序**
   - 标题放在最前
   - 说明段落在标题后
   - 前导句在说明段落后
   - 表格在前导句后

3. **检查表格完整性**
   - 确保没有多余符号（如 `#006`）
   - 表格分隔符 `| -------- |` 必须完整
   - 三行完整的数据行（表头 + 3 个材料行）

## 参考：正确的表格格式

```markdown
| 材料名称 | 内容要求                       | 提交形式    |
| -------- | ------------------------------ | ----------- |
| 阶段总结 | 已完成工作、遇到问题、后续安排 | PDF         |
| 实验记录 | 数据集、实验配置、结果截图     | 附件        |
| 论文提纲 | 章节结构、每章核心内容         | Word 或 PDF |
| 导师意见 | 导师确认签字或线上批注         | 扫描件      |
```

关键点：
- ✅ 分隔符行中的 `-` 数量要足够（通常至少 6 个）
- ✅ 每行的 `|` 对齐
- ✅ 没有多余符号或特殊字符嵌入
