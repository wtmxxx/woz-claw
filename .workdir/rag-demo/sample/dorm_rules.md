# 宿舍管理公约

为营造整洁、安全、安静的住宿环境，现对宿舍日常管理要求如下。

## 1. 作息要求

- 晚上 23:30 后保持安静，不得大声喧哗。
- 凌晨 0:00 后原则上不安排访客进入宿舍楼。
- 熄灯后禁止打游戏、播放外放视频或进行高分贝通话。

## 2. 安全要求

1. 严禁使用电炉、热得快、电热锅等大功率违规电器。
2. 严禁私拉电线、遮挡消防设施或堵塞安全出口。
3. 离开宿舍时请关闭空调、电扇和充电设备。
4. 发现异味、冒烟或漏水情况，应立即上报宿管老师。

## 3. 卫生要求

| 项目     | 频次 | 标准               |
| -------- | ---- | ------------------ |
| 地面清洁 | 每日 | 无明显垃圾、无积水 |
| 垃圾清运 | 每日 | 垃圾不堆放过夜     |
| 卫生间   | 每周 | 无异味、排水通畅   |

## 4. 违纪处理

若出现下列情况，将按学院规定处理：

（1）第一次违规，口头提醒并登记。
（2）第二次违规，书面通报并限期整改。
（3）造成安全事故的，移交学校相关部门处理。

## 5. 备注

宿舍管理不是“只要不出事就行”，而是要把风险控制在日常习惯里。越是看起来小的细节，越容易在学期后半段集中暴露问题。
