# 校园网升级通知

各位师生：

为了提升校园网稳定性与访问速度，信息化中心计划于本周六凌晨进行核心设备升级。升级期间，部分宿舍区、教学楼无线网络会出现短时中断，请提前做好资料保存与下载工作。

## 一、影响范围

- 东区宿舍 1-6 栋
- 教学楼 A、B、C
- 图书馆三层及以上区域

## 二、时间安排

| 时间          | 内容             |
| ------------- | ---------------- |
| 00:00 - 00:30 | 关闭部分认证服务 |
| 00:30 - 01:30 | 核心交换设备切换 |
| 01:30 - 02:00 | 联网测试与恢复   |

## 三、注意事项

1. 请提前保存在线文档、作业和考试资料。
2. 升级期间建议使用流量或有线网络备用方案。
3. 如遇断网，请勿反复重连，等待 5-10 分钟后再试。
4. 如有紧急情况，请联系信息化中心值班电话：8888。

答：不会。校园卡系统与本次升级网络段隔离，预计不受影响。
