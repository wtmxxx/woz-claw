# 图书馆借阅与座位使用须知

## 借阅规则

图书馆实行实名借阅制度。读者凭校园卡进入馆内，自助借还设备可用于普通图书借阅、续借与归还。

### 借阅期限

- 教师：60 天
- 本科生：30 天
- 研究生：45 天

### 超期处理

1. 超期图书按每日计算逾期费用。
2. 逾期超过 30 天的，暂停借阅权限。
3. 遗失图书需按原书价赔偿，特殊版本按馆内规定处理。

## 座位管理

为提高座位使用效率，图书馆实施预约与签到制度。

| 场景       | 规则                                 |
| ---------- | ------------------------------------ |
| 预约成功后 | 15 分钟内签到，否则自动释放          |
| 中途离开   | 30 分钟内未返回，座位失效            |
| 长时间占座 | 发现物品长时间无人看管，将按占座处理 |

## 温馨提示

请勿在馆内大声交谈、饮食或使用有明显噪音的设备。期末考试周座位紧张，建议同学们错峰到馆。
