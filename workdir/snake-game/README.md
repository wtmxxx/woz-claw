# 贪吃蛇游戏

一个使用 HTML、CSS 和 JavaScript 实现的经典贪吃蛇游戏。

## 功能特点

- 🐍 经典贪吃蛇玩法
- 🎮 使用方向键控制蛇的移动
- 🍎 吃到红色食物得分（每次+10分）
- 🚫 撞墙或撞到自己游戏结束
- 🔄 开始/重新开始按钮

## 文件结构

```
snake-game/
├── index.html    # 主页面
├── style.css     # 样式文件
├── game.js       # 游戏逻辑
└── README.md     # 说明文档
```

## 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

## 操作说明

1. 点击"开始游戏"按钮启动游戏
2. 使用键盘方向键（↑ ↓ ← →）控制蛇的移动
3. 吃到红色食物得分
4. 避免撞墙和撞到自己的身体
5. 游戏结束后点击"重新开始"按钮再次开始

## 技术实现

- **HTML5 Canvas**: 用于绘制游戏画面
- **CSS3**: 实现美观的界面布局和样式
- **JavaScript**: 实现游戏逻辑和交互控制
