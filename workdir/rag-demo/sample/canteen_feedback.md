# 食堂意见反馈记录

为持续改善校园餐饮服务，现收集 2026 年春季学期学生反馈。

## 常见反馈类别

- 菜品口味：偏咸、偏油、辣度不稳定
- 排队效率：午高峰拥堵明显
- 价格体验：部分窗口性价比不高
- 服务细节：餐具补充不及时、打包袋不足

## 调查摘录

1. “希望周三增加轻食套餐。”
2. “早餐区可以多增加两个收银窗口。”
3. “建议标明每道菜的辣度和过敏原。”
4. “晚餐时段最好延长 15 分钟。”

## 处理原则

| 问题类型 | 处理原则               |
| -------- | ---------------------- |
| 价格争议 | 核查公示与实际售价     |
| 食品安全 | 立即上报并暂停相关菜品 |
| 卫生问题 | 当日整改、次日复查     |

备注：反馈渠道包括线上问卷、班级意见箱和食堂服务台。
